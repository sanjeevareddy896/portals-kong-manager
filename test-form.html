Service Form Test

Test the ServicewithRouteForm component

How to test the form:

  • 1. Fill in the Service Name (required)
  • 2. Choose URL Configuration and enter a valid URL (e.g., https://api.example.com)
  • 3. Fill in the Route Host (required)
  • 4. Select at least one HTTP Method (required)
  • 5. Select at least one Protocol (required)
  • 6. Choose Preserve Host option (required)
  • 7. The Submit button should become enabled
  • 8. Check the browser console for validation logs
  • 9. For testing without real API calls, run: authHelper.setupForTesting() in console

Debug Information:

Open browser console (F12) to see validation logs and any errors.

Form validation status will be logged to console.
